# Exam Cheating Pattern Detection Dashboard 📊

An interactive Power BI dashboard to analyze online exam data and identify suspicious student behavior across 600+ records.

## Features

- KPI cards for Suspicion Score, Flag Count, and Score Anomalies
- Average Time per Question analysis
- Visual reports to identify outliers and patterns
- Built with Power Query and DAX for data cleaning and metric generation

## Tools Used

Power BI, Power Query, DAX, Microsoft Excel

## Dataset

600+ student exam records with behavioral metrics including response time, score patterns, and flag counts.

## How to View

1. Download the `.pbix` file
2. Open it in [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
